# photo-gallery

Follow the instruction in https://nodejs.org/en/download/ to install Node.js

Go to the photo-gallery directory, install all the dependencies we use in this project by

```
npm install
```


<br>

How to run it, you ask? Simply do
```
ionic serve
```
